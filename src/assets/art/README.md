# Art — designer asset library

Designer-authored illustration / brand art (app icon source, empty-state art, the *Molten Harmonics*
loader stills, marketing imagery). Keep editable SVG/source here; export rasters only where a platform
demands them (e.g. iOS app-icon PNGs for the Capacitor build).

Conventions:
- SVG where possible; theme-aware via CSS variables, not baked colours.
- App-icon master as a single high-res SVG → export the iOS/PWA sizes from it.
- Source of truth: the project Figma (add link here once it exists).
